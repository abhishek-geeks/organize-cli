# Organize CLI

Smart file organizer for automatic folder cleanup.

## Features (v1)
- Recursive file organization
- Category-based sorting
- Duplicate detection
- Dry-run preview
- Safe file moving

## Install

```bash
pip install git+https://github.com/<your-username>/organize-cli.git


organize ~/Downloads
organize --dry-run ~/Downloads
